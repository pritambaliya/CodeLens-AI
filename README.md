# CodeLens AI

> AI-Powered Code Review Platform that helps developers write cleaner, more efficient, and maintainable code.

🌐 **Live Demo:** https://codelens-ai-2v7l.onrender.com/

---

## 📖 Overview

CodeLens AI is a full-stack web application that analyzes source code using Artificial Intelligence and provides intelligent feedback on bugs, performance, security, best practices, and code quality.

It is designed to help students, developers, and teams improve their code faster by receiving instant AI-generated reviews without manual inspection.

---

## ✨ Features

- AI-powered code review
- Detects bugs and potential issues
- Performance optimization suggestions
- Security recommendations
- Time & Space Complexity Analysis
- Overall code quality rating
- AI Chat Assistant for programming help
- JWT Authentication with Secure Cookies
- Review history management
- Save favorite reviews
- Fully responsive modern UI
